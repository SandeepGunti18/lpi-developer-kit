## Overview

This project is an AI-based decision agent that helps students choose the best university and academic path based on their personal preferences.

The agent accepts user inputs such as:

- Interests
- Preferred course
- Budget
- Domestic or International study preference

It then generates structured recommendations with explainable outputs.

---

## LPI Tools Used

### 1. query_knowledge
Used to provide decision criteria such as curriculum quality, internships, placements, rankings, and ROI.

### 2. get_insights
Used to generate personalized recommendations based on budget and location preference.

---

## Example Use Case

Input:

- Interest: Software Development
- Course: BTech CSE
- Budget: ₹3000000
- Preference: International

Output:

- Recommended Countries: USA / UK / Singapore
- Next Steps:
  - Compare universities
  - Check scholarships
  - Apply before deadlines
